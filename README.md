# EVE Fury Reconstruction Demo v4.00

A demo-only MT5 XAUUSD momentum scalper reconstructed from the observable behaviour in Markflipper/Fury Flipper videos and screenshots.

Main EA: `mt5/EVE_Fury_Reconstruction_Demo_v4.00.mq5`

The build uses live tick momentum, one directional stop order, rapid same-direction continuation entries, and an individual broker-side SL and TP on every position. It is not the seller's original code and cannot be represented as an exact clone.

Start with 0.01 lot, 10 maximum positions, and a hedging demo account. Compile in MetaEditor before attaching it to XAUUSD M1.
