# Easy installation — v4.10

1. Turn Algo Trading OFF.
2. Remove v4.00 and clear any old EVE exposure.
3. Replace the entire GitHub repository with this package.
4. Railway must use Root Directory `railway` and start command `npm start`.
5. Wait for Railway version `4.1.0`.
6. Copy `mt5/EVE_Fury_Reconstruction_Demo_v4.10.mq5` into `MQL5/Experts`.
7. Compile it in MetaEditor. Continue only with 0 errors.
8. Add the Railway base URL to MT5's allowed WebRequest URLs.
9. Attach v4.10 to XAUUSD M1 on a hedging demo account.
10. Enter the Railway URL and BOT_TOKEN, then press OK.
11. Turn Algo Trading ON.
12. Confirm EA `4.10`, strategy `INDIVIDUAL_SL_TP_MOMENTUM_LADDER`, and magic `2407202641` on the dashboard.
