BEFORE REPLACING v1.00
1. Confirm the Momentum Burst dashboard shows no active position and no pending order.
2. Disable autonomous mode.
3. Remove EVE_Momentum_Burst_EA_v1.00 from the XAUUSD M1 chart.

GITHUB / RAILWAY
1. Replace the entire repository contents with this folder.
2. Keep Railway Root Directory: railway
3. Existing Railway variables remain correct:
   BOT_TOKEN=EVE-MOMENTUM-DEMO-2026
   AUTO_ENABLED=true
   DATA_DIR=/data
4. Wait for Railway to deploy v1.1.0.

MT5
1. Copy mt5/EVE_Momentum_Burst_EA_v1.10.mq5 into MQL5/Experts.
2. Open it in MetaEditor and press F7.
3. Continue only after MetaEditor reports 0 errors.
4. Attach it to the existing XAUUSD M1 chart.
5. In Inputs use the same Railway URL and BOT_TOKEN already used by v1.00.
6. Confirm Algo Trading is green.

SAFE DEFAULT BURST
- 3 x 0.01 initial positions
- maximum 5 positions
- maximum 0.05 total lots
- continuation orders add only while the basket is already profitable
- no martingale and no averaging into losses

This is a behavioural reconstruction from visible evidence. It is not Mark Flipper's private source code or guaranteed profitability.
